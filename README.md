Forget `npm` we will use `pnpm`. Install:
```bash
sudo npm install -g pnpm
```

Install packages:
```sh
pnpm install
```

Launch frontend:
```sh
pnpm run start
```

## Docker

Build locally for debug:
```sh
pnpm run build:docker
```
(Can be really long)

Run the built image:
```sh
pnpm run start:docker
```

## Quality

Testing:
```bash
pnpm run lint
```

Linting:
```bash
pnpm run test
```
